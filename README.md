# Vim first
###  vim use muscle not memory
There are 3 modes
  - ( esc ) Normal mode  : move  
  - ( i ) Insert mode    : type only 
  - ( v ) Visual mode    : select

# Command
```
esc
  :
    w - save
    q - quit
      q! - quit force

```
